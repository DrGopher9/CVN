Module 1: Cybersecurity

In competitions like CCDC, securing the perimeter is only half the battle. You have to understand how attackers think, how they break cryptography, and how they exploit bad code.

1. Cryptography: The Caesar Cipher

If an attacker intercepts your network traffic, encryption ensures all they see is garbage data. Your Challenge: An allied CCDC team sent you a distress signal, but it's encrypted. Slide the key to find the readable message.

2. Password Auditing: Brute Force

Hackers don't usually guess passwords; they use automated tools that can try millions of combinations per second. Your Challenge: Type a password below to see how long it would take a standard cracking rig to break it.

Estimated Time to Crack: Instantly

Pro Tip: Try adding a symbol, a number, or just making it really long (like a phrase).

3. Web Exploitation: SQL Injection

Sometimes the vulnerability isn't the password, but the website itself. Poorly written code can allow attackers to inject database commands directly into a login box.
